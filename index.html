ofxPiMapper

Open source projection mapping tool that runs on the Raspberry Pi

ofxPiMapper allows you to use less expensive hardware in order to help you focus on the things that matter. It's perfect for all kinds of media installations where one has to leave a computer running 24/7.

With Pi Mapper you can:

  • Use the Raspberry Pi or any other computer
  • Map images, videos as well as generative content made in openFrameworks
  • Have a separate media source for every mapping surface

You get:

  • Ability to use and adjust the ofxPiMapper source code to your needs
  • Possibility to use less expensive embedded platforms
  • Safety as there is a difference between a failing machine in worth of 1500 EUR and 100 EUR
